# ML11
Generic Numeric Analyzer & Predictor
Overview

This Python script allows you to analyze any numeric dataset and make simple predictions. You can upload a CSV file, visualize relationships between columns, and use Linear Regression to predict one numeric column from another.

Features

Upload CSV: Load any dataset in CSV format.

Column Selection: Select any two numeric columns for analysis.

Scatter Plot: Visualize the relationship between the selected columns.

Linear Regression Prediction: Predict one column from another.

RMSE Evaluation: Evaluate the prediction accuracy.

How to Use

Run the script in Google Colab.

Upload your CSV dataset when prompted.

Select the X-axis and Y-axis columns by number.

View basic statistics for the selected columns.

See a scatter plot of the two columns.

View a regression line predicting Y from X, along with RMSE.
